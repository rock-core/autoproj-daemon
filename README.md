# Autoproj::Daemon


## Installation

Run

~~~
autoproj plugin install autoproj-daemon
~~~

From within an autoproj workspace

## Usage

## Development

Install the plugin with a `--path` option to use your working checkout

~~~
autoproj plugin install autoproj-daemon --path /path/to/checkout
~~~
